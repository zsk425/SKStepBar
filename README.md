## SKStepBar

A very simple step view.

#### Example
```objc
- (void)viewDidLoad {
    [super viewDidLoad];

    self.stepBar.titles = @[@"Step 1",@"Step 2", @"Step 3",@"Step 4"];
    self.stepBar.tintColor = [UIColor lightGrayColor];
    self.stepBar.selectedColor = [UIColor orangeColor];
}
```
